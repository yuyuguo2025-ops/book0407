# AI 课堂协同系统

一个基于 Express + Socket.io 的实时课堂互动系统。

## ✨ 功能特性

- 📝 **座位签到系统** - 实时座位分配与签到
- 📚 **任务一：内容推荐** - 基于阅读标签的个性化书籍推荐
- 🔍 **任务二：协同过滤** - 基于同学阅读口味的智能推荐算法
- 💡 **任务三：实践创新** - 数据挖掘算法方案设计
- 🏛️ **班级方案广场** - 学生作品发布、浏览、点赞互动
- 👨‍🏫 **教师管理端** - 任务发布、作品管理、实时监控

## 🛠️ 技术栈

- **后端**: Node.js + Express.js
- **实时通信**: Socket.io
- **前端**: HTML + CSS + JavaScript
- **数据存储**: JSON 文件
- **可视化**: SVG 雷达图

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
npm start
```

然后访问：
- 学生端: http://localhost:3000
- 教师端: http://localhost:3000/teacher.html

## 🔄 协同过滤算法

基于余弦相似度计算同学间的阅读口味相似度，>50% 视为相似。
